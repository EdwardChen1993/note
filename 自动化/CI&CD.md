# CI/CD

## 概念

前端自动化是指前端代码的自动化构建、打包、测试及部署等流程。

**持续集成（Continuous Integration）**

**持续部署（Continuous Deployment）**

前端自动化通常与CI/CD流程相结合。



## 意义

+ 减少人为失误，提高软件质量
+ 效率迭代，便捷部署
+ 快速交付，便于管理
