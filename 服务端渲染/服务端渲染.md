# 服务端渲染



## 概念

**Server-Side Rendering（SSR）服务端渲染**是用户第一次请求/刷新页面时，由服务端响应HTML字符串，可以省去浏览器端首次渲染的工作，加快首屏显示速度。

**特点：**

+ 更好的seo
+ 更好的内容到达时间（time-to-content）



## 目前vue SSR方案

+ vue & vue-server-renderer 2.3.0+
+ nuxt.js 框架
